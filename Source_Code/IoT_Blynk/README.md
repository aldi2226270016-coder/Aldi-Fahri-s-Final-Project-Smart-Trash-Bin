# IoT Monitoring using Blynk

This folder contains the source code for the ESP32-based Smart Trash Bin with IoT monitoring using the Blynk platform.

## Features

- Automatic waste sorting
- Plastic, paper, and metal detection
- Real-time monitoring using Blynk
- LCD display
- Ultrasonic capacity monitoring
- Full-bin detection
- Error detection (waste jam)
- Standby mode

## Main File

- Smart_Trash_Blynk.ino

## Required Libraries

- WiFi.h
- BlynkSimpleEsp32.h
- ESP32Servo.h
- LiquidCrystal_I2C.h
- Wire.h
