# Datasheets

This folder contains datasheets for all electronic components used in this project.

- ESP32U
- E18-D80NK Infrared Sensor
- Inductive Proximity Sensor
- Capacitive Proximity Sensor
- HY-SRF05 Ultrasonic Sensor
- Servo Motor Hitec HS645-MG
- LCD 20x4 I2C
- DFPlayer Mini 
- Stepdown XL4015
- Stepdown XL4005
- HLK LD2410 Sensor
